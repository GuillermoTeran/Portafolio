Portafolio de Proyectos - Guillermo Teran

1. Análisis de Datos para Zuber: Comportamiento de viajes compartidos en Chicago

Resumen: Exploración y visualización del comportamiento de los usuarios y patrones de viajes en la ciudad de Chicago usando datos de Zuber. Se identificaron tendencias geográficas, temporales y de uso.

Herramientas: Python (Pandas, Matplotlib, Seaborn), Jupyter Notebook

2. Análisis de tarifas prepago para Megaline

Resumen: Estudio comparativo entre planes "Smart" y "Ultra" para entender el comportamiento de llamadas, mensajes y datos móviles, ayudando a definir una estrategia tarifaria óptima.

Herramientas: Python, Jupyter Notebook, Estadística Descriptiva e Inferencial

3. Análisis de venta de videojuegos y detección de patrones de éxito – Proyecto Ice Games

Resumen: Evaluación de datos de ventas para predecir el éxito de futuros lanzamientos. Se identificaron plataformas líderes, géneros preferidos y patrones estacionales en diferentes regiones.

Herramientas: Python, Pandas, Seaborn, Matplotlib

4. Priorización de hipótesis y análisis de pruebas A/B – Proyecto de E-commerce

Resumen: Evaluación de hipótesis con el framework ICE y análisis de dos pruebas A/B para medir el impacto de cambios en el sitio web sobre la conversión y el comportamiento de los usuarios.

Herramientas: Python, Scipy, Pandas, Estadística Inferencial, Visualización de Datos

5. Análisis de comportamiento de compra de clientes – Proyecto Instacart

Resumen: Estudio del patrón de compras de clientes para generar insights sobre hábitos, frecuencia, productos más demandados y segmentación del comportamiento del consumidor.

Herramientas: Python, SQL, Pandas, Visualización con Matplotlib/Seaborn

6. Análisis de eficiencia de marketing y comportamiento de clientes – Proyecto para Showz

Resumen: Se identificaron los canales de adquisición más rentables, comportamiento por segmento de usuarios y ROI de campañas de marketing.

Herramientas: Python, SQL, Estadística, Tableau

7. Predicción de cancelación de clientes y segmentación de usuarios – Proyecto para Model Fitness

Resumen: Se entrenaron modelos predictivos para identificar clientes con alta probabilidad de cancelar la suscripción. Se implementó una estrategia de retención basada en clustering de usuarios.

Herramientas: Python (Sklearn, Pandas), Machine Learning, Clustering K-Means

8. Caso de Estudio: Dashboard de Tendencias de Videos en YouTube

Resumen: Automatización de reportes de tendencias semanales en YouTube usando un dashboard en Tableau, enfocado en categorías más virales y distribución geográfica.

Herramientas: Tableau Public, Python (Pandas), PostgreSQL (simulado)

Ver Dashboard | Repositorio GitHub

9. Análisis de comportamiento de usuarios y test A/A/B en una app de productos alimenticios

Resumen: Análisis del embudo de conversión y evaluación de un experimento A/A/B para medir el impacto del rediseño en tipografía sobre la conversión de usuarios.

Herramientas: Python (Pandas, Scipy, Statsmodels), Visualización, Pruebas de hipótesis
